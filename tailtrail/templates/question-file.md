## Question Rules

- Keep each question specific.
- Use meaningful choices only.
- Make the final option `Other`.
- Include one recommended option after each question.
- Explain the recommendation briefly so the user can accept it, change it, or choose another option.
- Do not hide uncertainty; say when the recommendation depends on an assumption.
- Ask in a file for non-trivial ambiguity.
- After answers are filled in, check for missing, invalid, contradictory, or ambiguous responses.
