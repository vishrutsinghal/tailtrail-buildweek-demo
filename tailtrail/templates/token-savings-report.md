# TailTrail Token Savings Report

- Mode: `estimated | measured | unknown`
- Evidence level: `local_approximation | api_usage_metadata | missing_measured_telemetry`
- Baseline tokens: `0`
- TailTrail tokens: `0`
- Saved tokens: `0`
- Reduction: `0%`
- Claim guardrail: Estimated savings must not be presented as exact model/API token savings.

## Inputs

For estimated mode, list used context and avoided context with approximate token counts.

For measured mode, list provider/model usage records that include baseline total tokens and TailTrail total tokens.

## Safe Wording

- Estimated mode: say `estimated context reduction`, not exact savings.
- Measured mode: say `measured token reduction` only for records backed by model/API usage metadata.
- Missing telemetry: say exact token savings are unavailable.
