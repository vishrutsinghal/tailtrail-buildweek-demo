# TailTrail Assets

These assets are optional presentation and packaging polish for TailTrail.

They are original project assets created for this repository. They are not required for any TailTrail workflow, command, hook, skill, adapter, or local development behavior.

## Files

- `tailtrail-logo.svg`: full horizontal logo for README, internal demos, and marketplace-style listings.
- `tailtrail-mark.svg`: compact square mark for small cards, slide corners, or internal catalog thumbnails.

## Usage Rules

- Do not load these assets into routine coding prompts.
- Do not make TailTrail commands depend on these assets.
- Do not replace engineering docs with visual branding.
- Keep future assets small, original, and easy to audit.
- Do not add third-party logos, stock images, icon packs, or generated brand systems without explicit review.

## Provenance

These SVG files are original TailTrail project assets. No third-party source code, assets, or documentation are vendored here.
